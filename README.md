# Libro Libre

Libro Libre es un sitio web que permite a las personas acceder de manera gratuita a libros. La aplicación incluye un sistema de reservas en el que los usuarios deben iniciar sesión para almacenar sus datos y poder reservar libros para leer. Además, ofrece filtrado por género, autor o título específico.

## Características
- Acceso gratuito a libros.
- Sistema de reservas con inicio de sesión.
- Filtrado por género, autor o libro.
- Creación automática de cuentas con rol de usuario.
- Rol de administrador disponible mediante usuario preexistente en la base de datos o creación directa en la base de datos.

## Tecnologías utilizadas
- **Frontend:** React, JavaScript, HTML, CSS.
- **Backend:** Java, Spring Boot.
- **Dependencias y librerías:** Lombok, Hibernate, MSHR, Mave.
- **Base de datos:** SQLite.
- **Herramientas de prueba:** Postman.
